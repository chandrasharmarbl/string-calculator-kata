# String Calculator Kata

This repository contains the implementation of the String Calculator kata in Python.

## Behaviors Implemented

Tests are defined based on the requirements of the [String Calculator Kata](https://osherove.com/tdd-kata-1) by Roy Osherove. The test suite directly documents the supported behaviors, including:

- Basic operations with empty strings, single numbers, and multiple comma-separated values.
- Newline characters as valid delimiters alongside commas.
- Support for user-defined custom delimiters.
- Input validation to prohibit negative numbers (and consecutively report all negative inputs).
- Ignoring numbers greater than 1000.
- Tracking the frequency of calculator usage.
- Handling of long and multiple custom delimiters.
